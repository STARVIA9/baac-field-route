    🎯 เว็บนี้คืออะไร?

    เครื่องมือช่วย วางแผนเยี่ยมลูกค้าติดตามหนี้ + คำนวณเบี้ยเลี้ยง 160฿/วัน อัตโนมัติ + ออกรายงานส่งหัวหน้า

    ⚡ Quick Start — 5 ขั้นตอนหลัก

    1. เพิ่มลูกค้า แท็บ 👥 ลูกค้า → กด ➕ → กรอกชื่อ + ที่อยู่ + จิ้มแผนที่เพื่อเลือกตำแหน่ง → บันทึก 💡 ตั้งระดับความเสี่ยง (เขียว/เหลือง/แดง) + ประเภทหนี้ (ถึงกำหนด/ค้าง) + ชื่อเล่น + รูปถ่าย ด้วย 🗑️ ลบลูกค้า: กดปุ่ม 🗑️ ที่รายการ → ยืนยัน → ลบถาวร
    2. วางแผนเส้นทาง แท็บ 🧭 เส้นทาง → กด 🧮 คำนวณเส้นทาง → เลือกรถ → กด 💾 บันทึก 💡 หรือใช้ ⚡ เร็ว ถ้ามีไม่เกิน 10 ราย
    3. ออกเยี่ยม แท็บ 📋 เข้าพบ → กดที่ลูกค้า → เลือกผล → กด 📍 บันทึกพิกัด → 💾 บันทึก
    4. Sync อัตโนมัติ ข้อมูลจะซิงก์ขึ้น ☁️ Cloud ทุก 3 วินาที — หัวหน้าดูเรียลไทม์ได้
    5. สร้างรายงาน กด 📊 (header) → เลือกช่วงวันที่ + รถ → Export HTML/CSV/Share → ส่งหัวหน้า

    🎨 สีหมุด = ระดับความเสี่ยง

    • เขียว = ลูกค้าดี ติดตามง่าย
    • เหลือง = เริ่มมีปัญหา ยังติดต่อได้
    • แดง = มีปัญหามาก ติดต่อยาก/ไม่ได้
    • ❓ ไม่มีสี = ยังไม่ได้จัดระดับ (เข้าไปแก้ไขลูกค้า)

    📋 สถานะการเข้าพบ (เลือกตอนบันทึก)

    ✅ visitedเยี่ยมสำเร็จ · พบลูกค้านับ ✓
    💚 interestedลูกค้าสนใจนับ ✓

    🚫 not_interestedไม่สนใจไม่นับ
    💡 ต้องเลือก visited หรือ interested เท่านั้น → จึงจะนับเบี้ยเลี้ยง

    💰 เบี้ยเลี้ยง 160฿/วัน — นับยังไง?

    ใน 1 วัน เจ้าหน้าที่ต้องเยี่ยมลูกค้า:

    • ✅ ≥ 10 ราย หนี้ถึงกำหนด → ได้ 160฿
    • ✅ ≥ 5 ราย หนี้ค้าง → ได้ 160฿

    (ถึงเกณฑ์ใดเกณฑ์หนึ่งก็ได้ ไม่บวกกัน)

    📌 ตัวอย่าง:

    • จันทร์: เยี่ยม upcoming 12 + overdue 3 → ได้ 160฿ (ถึงเกณฑ์แรก)
    • อังคาร: เยี่ยม upcoming 7 + overdue 6 → ได้ 160฿ (ถึงเกณฑ์สอง)
    • พุธ: เยี่ยม upcoming 5 + overdue 2 → ได้ 0฿ (ไม่ถึง)

    📊 รายงาน — 3 ทางเลือกส่งหัวหน้า

    • 🖨️ HTML — เปิดพิมพ์ → Save PDF → ส่ง LINE/Email
    • 📊 CSV — เปิด Excel → แก้ไขได้ → ส่ง Email
    • 📱 Share — Copy text → Paste ใน Messenger/LINE
    💡 รายงานแสดงพิกัด GPS (กดลิงก์เปิด Google Maps ตรวจสอบได้) + ชื่อผู้บันทึก + เวลา

    💡 Tips ใช้งานจริง

    • 📌 ก่อนออก: วางแผนเส้นทาง + เช็คลูกค้า debtType ที่ต้องเยี่ยม
    • 📍 หน้างาน: กดบันทึก GPS ทุกครั้ง (ยืนยันว่าไปถึง)
    • 🌐 Offline ได้: แอพทำงานแม้ไม่มีเน็ต — sync อัตโนมัติเมื่อกลับมา
    • 👥 ทำงานเป็นทีม: ข้อมูลซิงก์ทุกเครื่อง — หัวหน้าดู realtime ได้
    • 📷 ถ่ายรูป: เก็บรูปลูกค้าไว้ช่วยจำ — ย่ออัตโนมัติไม่เปลืองเน็ต
